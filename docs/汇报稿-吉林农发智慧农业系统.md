# 吉林农发智慧农业管理平台 — 系统汇报

---

## 一、角色体系

系统按**集团—农场—一线**三级架构，预设五类角色，数据权限逐级收敛：

| 角色 | 视角 | 数据范围 |
|------|------|----------|
| 农发集团管理 | 全局管控 | 全集团 |
| 农垦集团管理 | 全局管控 | 全集团 |
| 土地资源公司 | 全局管控 | 全集团 |
| 农场管理员 | 农场级 | 单农场 |
| 一线操作员 | 地块级 | 个人负责地块 |

不同角色在**首页看板、地图图层、功能入口、数据可见范围**上均有差异化配置。

---

## 二、应用布局

APP 采用**底部三 Tab**导航结构：

| Tab | 名称 | 承载内容 |
|-----|------|----------|
| 首页 | 工作台 | KPI 看板、待办预警、农情报告、资讯视频 |
| 土地资源 | 一张图 | 卫星地图、地块图层、种植分布、物联网设备 |
| 我的 | 个人中心 | 角色切换、人员档案、告警通知、系统设置 |

---

## 三、功能模块

### 首页工作台
- KPI 指标看板（总面积、产值、设备在线率、种植进度、确权完成率）
- 待办任务与设备告警
- 遥感监测报告 + 气象专报
- 新闻资讯 + 农技视频

### 土地资源一张图
- 三类地块图层（宗地 / 高标准农田 / 盐碱地）
- 种植分布视图（统种 / 承租）
- 物联网设备点位标注
- 地块详情（基本信息、权证、承包、合同附件）

### 种植生产
- 种植规划 → 农艺配方 → 作业填报 → 产量预估与上报
- 统种地成本自动汇总（农资、种子、化肥、农药、农机、人工、灌溉）

### 物联网监控
- 8 类设备（土壤传感器、气象站、墒情站、摄像头、孢子捕捉仪、虫情测报灯、水肥一体机、智能控制终端）
- 实时数据 / 历史趋势 / 分级告警 / 远程控制

### 辅助功能
人员档案、告警中心、通知中心、每日资讯、视频讲座、天气概览
